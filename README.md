# nucleus-monorepo

## Install Dependencies
```bash
git clone https://github.com/NucleusNews/nucleus-monorepo.git
cd nucleus
npm install
```

## Environment Variables (Server)

```bash
cp apps/server/.env.example apps/server/.env
```

## Running (Development)

```bash
npm run dev:client
npm run dev:server
```

## Building
```bash
npm run build:client
npm run build:server
```
